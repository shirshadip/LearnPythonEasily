    3. What is interpreter?

  • An interpreter is a type of program that executes source code directly, one instruction at a time, instead of translating the entire program into machine code before execution.
  • some interpreted languages ----------> Python,Javascript,Shell


    • the difference in compiler and interpreter is that, compiler converts the High Level program to a machine Level code (Assembly Language). But for Assembly Language there is assembler which converts machine code to binary Language.
